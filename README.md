# 薄荷引擎 (Mint Engine)

薄荷引擎，方块空间使用基础3D薄荷引擎，渲染实物。

游戏使用：
方块空间
方块星际（星际方块）
宇宙方块

## 安装 (Install)
```
git clone https://github.com/uxnt/mint-engine.git
cd mint-engine
gradlew install
```
